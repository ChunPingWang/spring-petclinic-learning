## 🚀 立即開始使用

### 5 分鐘快速開始

```bash
# 1. 進入項目
cd spring-pet-learning

# 2. 編譯所有模組
mvn clean install

# 3. 啟動第一個模組
cd 01-basic-spring-boot
mvn spring-boot:run

# 4. 訪問應用
# 瀏覽器: http://localhost:8080/hello
```
